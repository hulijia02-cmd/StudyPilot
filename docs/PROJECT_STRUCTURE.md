# StudyPilot Project Structure

This repository is initialized for a Next.js App Router MVP.

## Core folders

- `app/`: Next.js routes, layouts, and route handlers.
- `components/`: Modular UI, layout, learning, and AI-facing components.
- `lib/`: Shared utilities, Supabase clients, and AI orchestration primitives.
- `types/`: Product, AI, and database TypeScript contracts.
- `supabase/`: Database migration files for Supabase.
- `docs/`: Product and engineering notes.

## First-stage scope

- Landing and learning cockpit screens.
- Goal onboarding and AI diagnosis placeholders.
- Supabase client/server setup.
- Health API and diagnosis API skeleton.
- Vercel deployment settings.
